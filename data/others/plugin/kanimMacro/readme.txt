■これは何ですか
――――――――――――――――――――――――――――――――――――――――――
　キーフレームアニメ定義とマクロのセットです。
　簡単な記述でキーフレームアニメを使用することができます。

――――――――――――――――――――――――――――――――――――――――――
■つかいかた
――――――――――――――――――――――――――――――――――――――――――
　プロジェクトファイルの data > others > plugin フォルダに《kanimMacro》フォルダを配置し
　シナリオファイル最初の方に[ plugin name="kanimMacro" ] と記述してください。
　プラグインが有効となり独自のマクロが使用できるようになります。
　マクロの種類については同梱の「マクロ一覧.txt」をごらんください。

　また、下記のURLに動作サンプルをアップしております。
　http://ko10panda.html.xdomain.ne.jp/kanimmacro_sample/index.html
